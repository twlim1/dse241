Instruction to run EDA python notebook
1. Navigate to root directory
2. Start jupyter notebook by typing "jupyter notebook"
3. Run eda.ipynb

Instruction to start the vis
1. Start a terminal
2. Change the working location of the terminal to the current directory
3. Start a python server by typing **python final.py**
4. Open the following URL (http://127.0.0.1:8050/) in a browser
